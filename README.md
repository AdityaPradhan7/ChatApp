# ChatApp (MERN, Socket.IO)

A simple realâ€‘time chat app with JWT auth, MongoDB Atlas persistence, and Socket.IO messaging.

## Features
- Email/password signup & login with JWT
- Realtime one-to-one chat over Socket.IO
- Conversation history and user list from MongoDB
- Protected routes on the client; localStorage token storage
- MongoDB Atlas for persistence

## Tech Stack
- Frontend: React (CRA), Tailwind CSS, Socket.IO client
- Backend: Node.js, Express, Socket.IO, JWT, bcryptjs, Mongoose
- Database: MongoDB Atlas
- Dev tools: nodemon

## Prerequisites
- Node.js 16+ and npm
- MongoDB Atlas cluster (or local Mongo)
- Two terminals (one for server, one for client)
